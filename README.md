# hexo-theme-Tech

Tech 是一个简洁、高效、且具有赛博朋克感的 Hexo 主题。

## 安装

```bash
git clone https://github.com/your-username/hexo-theme-Tech.git themes/Tech
```

## 配置

修改 Hexo 根目录下的 `_config.yml`：

```yaml
theme: Tech
```

## 特性

1. **三段式布局**：左侧分类、中间列表、右侧正文。
2. **实时搜索**：支持按标题、标签、作者进行快速检索。
3. **响应式设计**：完美适配移动端。
4. **内置交互**：包含赛博故障效果、鼠标交互音效等。
5. **极简体验**：专注于内容阅读。
